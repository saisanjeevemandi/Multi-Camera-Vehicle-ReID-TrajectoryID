# Dataset Information

This project uses the VeRi-776 Vehicle Re-Identification dataset for evaluation.

Due to licensing restrictions, the dataset is not included in this repository.
Please download it from the official source:

[https://vehiclereid.github.io/VeRi/](https://service.tib.eu/ldmservice/dataset/veri-776)

After downloading, organize your data as follows:

data/
├── video1.mp4
├── video2.mp4
└── sample/
    └── sample_frame.jpg

For quick testing, you may use any two traffic surveillance videos.
